# QShare

A simple file sharing app for Windows 10/11 -> Mobile.

# Software Requirements

Download an install Python:

[]()https://www.python.org/downloads/

# Module Requirements

```console
pip install pyqrcode
pip install pypng
pip install flask
pip install pyinstaller
```

# Build App
```console
pyinstaller QShare.spec
```
